# QhumanC2
This is QhumanC2 1.0 version. Starting...

Qhuman是一款多人协作的后渗透工具，拥有更严格，更精细的控制，设计上不是基于易操作，傻瓜型的一款工具，更多的操作请查看的文档。

Ver 1.0.0: 
  基本架构设计完成，全平台支持，支持的功能：
  1. 命令控制
  2. 日志记录
  3. 监听服务
  4. 代理服务
  5. 自定义扩展
  6. 全局控制
  7. shellcode，无文件支持
